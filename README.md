## Installation

To install [EthereumJS TestRPC](https://github.com/ethereumjs/testrpc).

```
$ npm i testrpc -g
```

then, execute TestRPC

```
$ testrpc
```

## Truffle Commands

| Process  |     Command    |
|----------|:--------------|
| Compile  | `$ truffle compile`  |
| Migrate  | `$ truffle migrate`  |
| Test     | `$ truffle test`     |
| Test with Coverage | `$ npm run cover`     |
